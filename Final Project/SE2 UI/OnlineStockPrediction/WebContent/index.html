Group-13's Final Project for ECE-568 Spring 2015 - Software Engineering Web Application

Highlights of our project

Web Services on Amazon AWS

Access to our web services would be required by the persons doing the development for the front end and the persons developing the prediction strategies. To make each personâ€™s work independent of each other, we decided to host our PostgreSQL database and web services on the Amazon cloud as it would help in true distributed development.

Featuring Google Trends

As a fun feature the user can view the google trends chart for the past 5 years for any stock. The idea behind this feature is that, a high trending value for any name on Google is directly proportional to the popularity of the topic(interest of the topic over time). This concept can be made use as a factor for weighting the buy/sell/hold decision by the user.

Good Recommendation Policy

As mentioned above the application provides recommendations to the user whether he should buy, sell or hold on to the stock. This decision is made using the predicted stock value using the long-term prediction algorithms. Google trend provides give a fair indication of the popularity of the company under review. Usually the more a company is searched, the popular it is and that makes it price to rise in the future.

UI Twitter BootStrap and Jersey Framwork

We have used Twitter Bootstrap for UI development. Jersey RESTful Web Services framework is open source, production quality, framework, for developing RESTful Web Services in Java that provides support for JAX-RS APIs and serves as a JAX-RS (JSR 311 & JSR 339) Reference Implementation.
